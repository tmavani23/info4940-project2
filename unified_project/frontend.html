Quick start

  • Share a memory or feeling in Chat (text, image, or audio).
  • Explore with follow-up questions: use template chips to reply.
  • Create by saying "help me implement" or similar when ready.
  • Refine in the Code panel: Run, Save Version, and use Version History.
  • Edit the Profile panel to adjust emotion or artistic direction, then Save Version.

How it works

Step 1: Share
In the Chat panel, describe a memory or feeling in your own words. You can also attach an image (📷) to show a mood or scene, or record audio (🎤) to describe it verbally.

Step 2: Explore
The assistant will ask follow-up questions to understand your emotions. Use the template chips above the input to jump-start replies (e.g., "The setting was…", "2–3 emojis that fit…").

Step 3: Create
When you're ready, say something like "help me implement" or "I have a good sense of the emotion." The assistant will generate p5.js code that appears in the Code panel.

Step 4: Refine
Edit the code directly in the Code panel. Click Run to preview changes. Click Save Version to keep a snapshot of your current code and profile. Use Version History to browse and restore earlier versions.

Profile panel

The Profile panel shows the assistant's understanding of your emotions and artistic direction. You can edit it directly: Emotion Profile (the assistant's summary of your emotions) and Artistic Profile (the visual direction for your sketch). Your edits are saved when you click Save Version in the Code panel—that creates a new version that includes your profile changes.

Version history

Click View Version History to open the edit history graph. You can browse all saved versions and how they connect, click a version to preview its code and profile without restoring, and use Restore to this version to bring back that version's code and profile. Use this to try different directions and return to earlier ideas.

Tips

  • You can switch between exploring emotions and generating code at any time.
  • The Profile panel shows the assistant's confidence in understanding your emotions.
  • If something feels off, say so—the assistant is designed to correct and iterate.
  • When the assistant proposes a new visual direction, you can Accept, Modify, or ask for More options.
